# Online Artisan Store - DBMS Project (ArtisanDB)

## Project Description
The Online Artisan Store is a complete relational database system designed for an e-commerce marketplace where artisans can list handmade products and clients can browse, order, and purchase items. The system also manages payments, deliveries, and customer reviews using a fully normalized database structure.

## Key Features
- User Management (Clients & Artisans)
- Product and Category Management
- Order Processing System
- Payment Tracking System
- Delivery Management System
- Product Reviews and Ratings
- SQL Triggers for automation
- Stored Procedures for business logic

## Database Features
- Fully normalized design (1NF, 2NF, 3NF)
- Primary and Foreign Key constraints
- Composite and multivalued attribute handling
- Automated processes using triggers
- Stored procedures for order handling
- Relational schema design
